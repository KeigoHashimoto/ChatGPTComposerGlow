# プライバシーポリシー

施行日: 2026年6月23日

ChatGPT Composer Glow は、ChatGPT のWeb画面に控えめな視覚的ステータス表示を追加するChrome拡張機能です。

English summary: ChatGPT Composer Glow is a Chrome extension that adds subtle visual state cues to the ChatGPT web interface.

## データの収集

ChatGPT Composer Glow は、個人情報を収集、保存、販売、共有、送信しません。

この拡張機能は、閲覧履歴、ChatGPTのメッセージ、プロンプト、回答、アカウント情報、その他ページ内の内容を、開発者または外部サーバーへ送信しません。

English summary: The extension does not collect, store, sell, share, or transmit personal information or page content.

## ローカルでのページアクセス

この拡張機能は、以下のChatGPTドメインでのみ動作します。

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`

入力欄の光、送信ボタンのアニメーション、回答待機中の表示、サイドバーのハイライトなどを適用するために、ブラウザ内でページの状態をローカルに読み取ります。この処理はユーザーのブラウザ内だけで行われます。

English summary: Page state is processed locally in the user's browser only on supported ChatGPT pages.

## 権限

ChatGPT Composer Glow は、対応するChatGPTページにローカルのJavaScriptとCSSを適用するために必要なコンテンツスクリプトのアクセス以外のChrome拡張機能権限を要求しません。

English summary: The extension does not request permissions beyond the content script access needed for supported ChatGPT pages.

## 第三者サービス

ChatGPT Composer Glow は、アクセス解析、広告ネットワーク、トラッキングピクセル、リモートコード、外部APIを使用しません。

English summary: The extension does not use analytics, ads, tracking pixels, remote code, or external APIs.

## データの保持

ChatGPT Composer Glow はデータを収集しないため、ユーザーデータを保持しません。

English summary: Because no data is collected, no user data is retained.

## ポリシーの変更

このプライバシーポリシーは、拡張機能の動作が変更された場合に更新されることがあります。重要な変更がある場合は、更新版を公開する前にこの文書へ反映します。

English summary: This policy may be updated if the extension's behavior changes.

## お問い合わせ

プライバシーに関する質問は、Chrome ウェブストアの掲載ページに記載されたサポート窓口から開発者へお問い合わせください。

English summary: For privacy questions, contact the developer through the support channel listed on the Chrome Web Store listing.
