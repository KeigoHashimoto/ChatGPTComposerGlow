# Privacy Policy

Effective date: June 23, 2026

ChatGPT Composer Glow is a Chrome extension that adds subtle visual state cues to the ChatGPT web interface.

## Data Collection

ChatGPT Composer Glow does not collect, store, sell, share, or transmit personal information.

The extension does not send browsing history, ChatGPT messages, prompts, responses, account information, or any other page content to the developer or to any external server.

## Local Page Access

The extension runs only on the following ChatGPT domains:

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`

The extension reads page state locally in the browser to determine when to apply visual effects, such as composer glow, send-button animation, response waiting state, and sidebar highlighting. This processing happens only in the user's browser.

## Permissions

ChatGPT Composer Glow does not request Chrome extension permissions beyond the content script access needed to apply local JavaScript and CSS to supported ChatGPT pages.

## Third-Party Services

ChatGPT Composer Glow does not use analytics, advertising networks, tracking pixels, remote code, or external APIs.

## Data Retention

Because ChatGPT Composer Glow does not collect data, it does not retain user data.

## Changes to This Policy

This privacy policy may be updated if the extension's behavior changes. Any material changes should be reflected in this document before publishing an updated version.

## Contact

For privacy questions, contact the extension developer through the support channel listed on the Chrome Web Store listing.
